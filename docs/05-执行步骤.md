# 执行步骤

> 本文件记录项目的分步实施计划。每一步都是独立可验证的里程碑。

## 第 1 步：项目脚手架 + 首页框架

**目标**：能启动前后端，首页展示倒计时和开始学习按钮。

### 任务清单

- [ ] 创建根 `package.json`（workspace 管理）
- [ ] 初始化 `client/`（Vite + React 项目）
  - 安装依赖：react, react-dom, react-router-dom, react-markdown, remark-gfm
  - 配置 vite.config.js（代理 API 到后端）
- [ ] 初始化 `server/`（Express 项目）
  - 安装依赖：express, cors, better-sqlite3
  - 创建 `server/index.js`（入口 + CORS + 静态文件）
  - 创建 `server/config.js`（端口、路径配置）
- [ ] 创建 `server/db/database.js`（SQLite 连接 + 自动建表）
- [ ] 创建 `server/db/migrations.js`（全部 CREATE TABLE 语句）
- [ ] 创建 `server/routes/countdown.js`（倒计时 GET/PUT API）
- [ ] 创建 `content/` 和 `data/` 目录
- [ ] 创建 `client/src/App.jsx`（React Router 路由配置）
- [ ] 创建 `client/src/components/layout/AppLayout.jsx`（主布局容器）
- [ ] 创建 `client/src/components/layout/TopBar.jsx`（4 Tab 导航）
- [ ] 创建 `client/src/pages/HomePage.jsx`（首页）
- [ ] 创建 `client/src/components/home/CountdownCard.jsx`（倒计时卡片）
- [ ] 创建 `client/src/components/home/FeatureList.jsx`（特性列表）
- [ ] 创建 `client/src/styles/globals.css`（CSS 变量 + 全局样式）
- [ ] 创建 `client/src/utils/api.js`（API 封装）
- [ ] 创建 `client/src/utils/constants.js`（常量定义）

### 验证标准

1. `node server/index.js` 启动无报错
2. `cd client && npm run dev` 启动前端无报错
3. 首页显示倒计时卡片（默认显示示例数据）
4. 点击倒计时可编辑名称和时间
5. 点击「开始学习」按钮跳转到 `/modules`
6. TopBar 的 4 个 Tab 可切换页面

### 涉及的关键文件

```
新建:
  package.json, client/, server/,
  content/, data/, client/src/{App.jsx,pages/HomePage.jsx,
  components/{layout/{AppLayout,TopBar},home/{CountdownCard,FeatureList}},
  styles/globals.css, utils/{api,constants}.js}
```
