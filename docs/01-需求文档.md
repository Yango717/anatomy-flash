# 解剖学学习复习软件 — 需求文档

## 1. 项目概述

| 项 | 内容 |
|---|------|
| 产品名称 | 解剖闪背 |
| 目标用户 | 大二医学生，准备解剖学期末考试 |
| 核心价值 | 系统化五阶段学习流，从识记到测试到错题强化 |
| 技术栈 | React (Vite) + Node.js (Express) + SQLite |
| 内容管理 | JSON / Markdown 文件 |
| 目标平台 | 手机端优先，响应式适配桌面 |

## 2. 用户旅程

```
首页（倒计时+开始学习）
  → 模块选择页（9大模块卡片）
    → 小节目录页（树形细分目录）
      → 学习流页面（五阶段循环）
        → Phase 1: 学习基本知识
        → Phase 2: 填空小测验
        → Phase 3: 自动回顾复习
        → Phase 4: 正式测试（错题入本）
        → Phase 5: 真题测试 [可选]
```

## 3. 功能需求

### 3.1 首页 (HomePage)
- **考试倒计时卡片**：显示目标日期的剩余天数和时分秒
- **倒计时名称可编辑**：如"距离解剖学期末考试"
- **倒计时目标时间可设定**
- **「开始学习」主按钮**：跳转到模块选择页
- **应用特性展示区**：简介列表

### 3.2 模块选择页 (ModulesPage)
- 展示解剖学 **9 大模块**为 2 列卡片网格
- 每个卡片显示：模块名、包含小节数、掌握率百分比
- **单击卡片直接进入**该模块的小节目录页

### 3.3 小节目录页 (SectionsPage)
- 展示当前模块下所有小节的树形目录
- 支持折叠/展开每个小节分组
- 每个最小学习单元显示当前阶段进度徽标
- 单击单元进入五阶段学习流

### 3.4 五阶段学习流

#### Phase 1 — 学习基本知识
- 展示内容类型：定义、位置、形态、结构、功能、临床意义、图谱
- Markdown 渲染 + 图片查看器
- 用户阅读后点击"完成学习"进入下一阶段

#### Phase 2 — 小测验（填空题）
- 题型：填空题（`___` 替换为输入框）
- 支持 hint 提示
- 提交后评分，错误记录到 weak_points（**不进错题本**）
- 全对则自动跳过 Phase 3

#### Phase 3 — 自动回顾
- 根据 Phase 2 错误自动生成回顾内容
- 每条回顾包含：原题、错答、正解、相关原文摘录
- 完成回顾后解锁 Phase 4

#### Phase 4 — 正式测试
- 题型：选择题、判断题、名词解释
- 名词解释支持关键词匹配评分
- **做错的题目进入错题本**（核心评估依据）

#### Phase 5 — 真题测试 [可选]
- 往年期末真题
- 可选择立即做或稍后做
- 可选是否也将错题加入错题本

### 3.5 错题本 (ErrorBook)
- 列表展示所有 Phase 4 错题
- 支持按模块/题型筛选
- 每道错题有掌握度等级 (0-3)
- 支持重测、提升掌握度、手动标记已解决
- 间隔重复调度（next_review_due）

### 3.6 设置/更多 (SettingsPage)
- 编辑倒计时名称和目标时间
- 数据重置选项
- 关于信息

## 4. 非功能需求

- **手机端优先**：主要使用场景为手机浏览器
- **离线友好**：静态内容可缓存，核心功能在网络恢复后可用
- **性能**：首屏加载 < 2s，页面切换流畅
- **数据安全**：学习进度本地存储，不上传服务器

| 序号 | 模块名称 | 主要内容 |
|------|---------|---------|
| 1 | 运动系统 | 骨学、关节学、肌学 |
| 2 | 消化系统 | 消化管、消化腺 |
| 3 | 呼吸系统 | 鼻、喉、气管、肺、胸膜 |
| 4 | 泌尿系统 | 肾、输尿管、膀胱、尿道 |
| 5 | 生殖系统 | 男/女性内、外生殖器 |
| 6 | 循环系统 | 心、动脉、静脉、淋巴 |
| 7 | 感觉器 | 视器、前庭蜗器 |
| 8 | 神经系统 | 中枢神经、周围神经 |
| 9 | 内分泌系统 | 甲状腺、垂体、肾上腺等 |
