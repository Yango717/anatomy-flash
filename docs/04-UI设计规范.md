# UI 设计规范

## 1. 设计原则

- **医学专业风**：医学蓝色主调 + 深色背景 + 白色卡片，沉稳专业
- **手机优先**：主要使用场景为手机竖屏
- **清晰的信息层级**：深色底衬托白色内容卡片，层次分明
- **一致的交互模式**：同类操作使用相同的交互方式

## 2. 配色方案

```css
:root {
  /* === 主色系 === */
  --color-primary: #1e6b9b;          /* 医学蓝 - 主按钮、活跃状态 */
  --color-primary-light: #2980B9;     /* 浅蓝 - hover 状态 */
  --color-primary-bg: #E3F2FD;        /* 淡蓝背景 - 标签、高亮 */

  /* === 辅助色系 === */
  --color-secondary: #00838F;         /* 深青 - 次要按钮 */
  --color-secondary-light: #0097A7;
  --color-secondary-bg: #E0F7FA;

  /* === 中性色 === */
  --color-bg: #151d28;               /* 深色页面背景 */
  --color-bg-subtle: #F3F4F6;         /* 卡片内部浅色对比背景 */
  --color-card: #FAFBFC;             /* 微灰白卡片 */
  --color-border: #E2E4E7;           /* 边框线 */
  --color-divider: #EEEEF0;          /* 分割线 */

  /* === 文字色 === */
  --color-text-primary: #18181b;     /* 主文字 */
  --color-text-secondary: #52525b;   /* 次要文字 */
  --color-text-hint: #8b8b8f;        /* 提示文字 */
  --color-text-inverse: #FFFFFF;     /* 反色文字（深色背景上）*/

  /* === 状态色 === */
  --color-success: #1e6b9b;           /* 成功/正确 */
  --color-error: #DC2626;            /* 错误 */
  --color-warning: #D97706;          /* 警告 */
  --color-info: #2563EB;             /* 信息 */
}
```

## 6. 组件规范

### 6.1 按钮

| 类型 | 样式 | 使用场景 |
|------|------|---------|
| Primary | 蓝底白字，圆角 12px | 主要操作（开始学习、提交）|
| Secondary | 青底白字 | 次要操作（深度学习入口）|
| Outline | 蓝边蓝字，透明底 | 辅助操作 |
| Ghost | 无边框，蓝字 | 文字链接类操作 |
| Danger | 红底白字 | 危险操作（删除、重置）|

**尺寸：**
- 默认高度：48px（触控友好）
- 小按钮：36px
- 大按钮：56px（首页主按钮）

### 6.2 卡片

- 背景：白色
- 圆角：16px
- 内边距：16-20px
- 阴影：--shadow-md
- 间距：卡片间 12px

### 6.3 Tab 导航（顶部）

- 高度：48px
- 选中态：蓝色文字 + 底部蓝色下划线 (2px 高)
- 未选中：灰色文字
- 4 个 Tab 等宽分布

### 6.4 徽标/标签 (Badge)

- 圆角：full（胶囊形）
- 内边距：4px 10px
- 字号：12px
- 阶段颜色：
  - Phase 0 (未开始): 灰色 #CCC
  - Phase 1 (已学习): 蓝色 #3498DB
  - Phase 2 (已测验): 黄色 #F1C40F
  - Phase 3 (已回顾): 青色 #1ABC9C
  - Phase 4 (已测试): 紫色 #9B59B6
  - Phase 5 (已完成): 蓝色 #1A5276

### 6.5 进度条

- 高度：8px
- 圆角：full
- 背景：浅灰 #EEE
- 填充：蓝色 #1A5276

## 7. 页面布局

### 7.1 首页布局

```
TopBar (固定顶部, 48px)
  ↓
内容区 (可滚动)
  ├─ 倒计时卡片 (居中, 最大宽度 360px)
  ├─ 开始学习按钮 (全宽, 最大宽度 360px)
  └─ 特性列表 (图标+文字, 2列网格)
```

### 7.2 模块选择页布局

```
PageHeader (返回+标题)
  ↓
内容区 (可滚动)
  └─ 模块网格 (CSS Grid, 2列等宽, gap 12px)
     └─ ModuleCard × 9
```

### 7.3 小节目录页布局

```
PageHeader (返回+模块名+进度)
  ↓
内容区 (可滚动)
  └─ SectionTree (折叠列表)
     └─ SectionGroup × N (可折叠)
        └─ UnitItem × N (可点击)
```

### 7.4 学习流页面布局

```
PhaseIndicator (阶段步骤条, 固定顶部)
  ↓
内容区 (根据 current_phase 渲染不同子组件)
  ↓
ActionBar (固定底部, 操作按钮)
```

## 8. 断点与响应式

| 断点 | 宽度 | 设备 | 布局变化 |
|------|------|------|---------|
| 默认 | <= 480px | 手机 | 单列，底部无额外导航 |
| md | 481-768px | 平板 | 侧边栏变为滑出遮罩 |
| lg | >= 769px | 桌面 | 左侧固定侧边栏 + 主内容区 |

## 9. 动效规范

- **页面切换**: fade 或 slide（200ms ease-out）
- **按钮点击**: scale(0.98) 反馈（100ms）
- **卡片出现**: translateY(8px) → 0（300ms ease-out）
- **展开/折叠**: height transition（250ms ease）
- **避免**: 旋转、弹跳、闪烁等花哨效果
