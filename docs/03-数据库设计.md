# 数据库设计

## 概述

数据库引擎：SQLite 3
数据库文件路径：`data/anatomy.db`
初始化方式：首次启动 server 时自动建表

---

## 表结构

### users — 用户表

单用户应用，预留多用户扩展。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 用户ID |
| username | TEXT | NOT NULL UNIQUE | 用户名 |
| display_name | TEXT | | 显示名称 |
| created_at | TEXT | DEFAULT now | 创建时间 |
| settings_json | TEXT | DEFAULT '{}' | 设置(JSON) |

### settings — 设置表

键值对存储，用于倒计时等用户偏好。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| key | TEXT | PRIMARY KEY | 设置键名 |
| value | TEXT | NOT NULL | 设置值(JSON或字符串) |
| updated_at | TEXT | DEFAULT now | 更新时间 |

**预设键值：**

| key | value 示例 | 说明 |
|-----|-----------|------|
| countdown_name | "距离解剖学期末考试" | 倒计时显示名称 |
| countdown_target | "2026-07-15T09:00:00" | 目标日期(ISO 8601) |

### unit_progress — 单元学习进度

追踪每个最小学习单元所处的阶段。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | 记录ID |
| user_id | INTEGER | NOT NULL FK→users | 用户ID |
| unit_id | TEXT | NOT NULL | 单元ID（如 sub-01-01-01-part-概述） |
| current_phase | INTEGER | DEFAULT 0 | 当前阶段 (0-6) |
| phase_1_completed_at | TEXT | | Phase1 完成时间 |
| phase_2_completed_at | TEXT | | Phase2 完成时间 |
| phase_3_completed_at | TEXT | | Phase3 完成时间 |
| phase_4_completed_at | TEXT | | Phase4 完成时间 |
| phase_5_completed_at | TEXT | | Phase5 完成时间 |
| last_accessed_at | TEXT | DEFAULT now | 最后访问时间 |

**唯一约束：**(user_id, unit_id)

**阶段值含义：**

| 值 | 含义 |
|----|------|
| 0 | 未开始 |
| 1 | 已学习 (Phase 1 完成) |
| 2 | 已测验 (Phase 2 完成) |
| 3 | 已回顾 (Phase 3 完成) |
| 4 | 已测试 (Phase 4 完成) |
| 5 | 已真题 (Phase 5 完成) |
| 6 | 全部完成 |

### quiz_attempts — 小测验作答记录

Phase 2 填空题的答题记录。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | |
| user_id | INTEGER | NOT NULL FK | |
| unit_id | TEXT | NOT NULL | 所属单元 |
| question_id | TEXT | NOT NULL | 题目ID (qz-xxx) |
| question_type | TEXT | NOT NULL | 题型 (fill_blank) |
| user_answer | TEXT | NOT NULL | 用户答案(JSON数组) |
| correct_answer | TEXT | NOT NULL | 正确答案(JSON数组) |
| is_correct | INTEGER | DEFAULT 0 | 是否全对 (0/1) |
| blanks_wrong | TEXT | | 错误的空位索引(JSON数组) |
| score | REAL | DEFAULT 0 | 得分比例 (0.0-1.0) |
| created_at | TEXT | DEFAULT now | |

### weak_points — 弱知识点

从 quiz_attempts 中派生，Phase 2 的错误记录（**不入错题本**）。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | |
| user_id | INTEGER | NOT NULL FK | |
| unit_id | TEXT | NOT NULL | |
| question_id | TEXT | NOT NULL | |
| related_content | TEXT | | 关联的知识点类别（如"定义"）|
| wrong_count | INTEGER | DEFAULT 1 | 答错次数 |
| last_wrong_at | TEXT | DEFAULT now | 最近答错时间 |
| reviewed | INTEGER | DEFAULT 0 | 是否已回顾 (0/1) |
| reviewed_at | TEXT | | 回顾时间 |

**唯一约束：**(user_id, unit_id, question_id)

### test_attempts — 正式测试作答记录

Phase 4 测试的答题记录。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | |
| user_id | INTEGER | NOT NULL FK | |
| unit_id | TEXT | NOT NULL | |
| question_id | TEXT | NOT NULL | 题目ID (tst-xxx) |
| question_type | TEXT | NOT NULL | multiple_choice / true_false / term_explanation |
| user_answer | TEXT | | 用户答案 |
| correct_answer | TEXT | NOT NULL | 正确答案 |
| is_correct | INTEGER | DEFAULT 0 | 是否正确 (0/1) |
| score | REAL | DEFAULT 0 | 得分 |
| max_score | REAL | DEFAULT 1 | 满分 |
| created_at | TEXT | DEFAULT now | |

### error_book — 错题本

**核心评估依据**。来自 Phase 4 中答错的题目。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | |
| user_id | INTEGER | NOT NULL FK | |
| unit_id | TEXT | NOT NULL | |
| unit_path | TEXT | | 人类可读路径："运动系统 > 骨学 > 概述" |
| question_id | TEXT | NOT NULL | |
| question_type | TEXT | NOT NULL | |
| question_stem | TEXT | NOT NULL | 题目快照 |
| user_answer | TEXT | | 用户答案快照 |
| correct_answer | TEXT | NOT NULL | 正确答案 |
| explanation | TEXT | | 解析 |
| source_test_attempt_id | INTEGER | FK→test_attempts | 来源测试记录 |
| mastery_level | INTEGER | DEFAULT 0 | 掌握等级 (0-3) |
| times_reviewed | INTEGER | DEFAULT 0 | 已复习次数 |
| next_review_due | TEXT | | 下次复习截止(间隔重复) |
| is_resolved | INTEGER | DEFAULT 0 | 是否已解决 (0/1) |
| resolved_at | TEXT | | 解决时间 |
| created_at | TEXT | DEFAULT now | |
| updated_at | TEXT | DEFAULT now | |

**掌握度等级：**

| 值 | 含义 |
|----|------|
| 0 | 新错题，未复习 |
| 1 | 已复习 1 次 |
| 2 | 已复习 2 次 |
| 3 | 已掌握 |

### final_exam_attempts — 真题作答记录

Phase 5 真题测试的答题记录。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | |
| user_id | INTEGER | NOT NULL FK | |
| unit_id | TEXT | NOT NULL | |
| question_id | TEXT | NOT NULL | 题目ID (fe-xxx) |
| question_type | TEXT | NOT NULL | |
| user_answer | TEXT | | |
| correct_answer | TEXT | NOT NULL | |
| is_correct | INTEGER | DEFAULT 0 | |
| score | REAL | DEFAULT 0 | |
| add_to_error_book | INTEGER | DEFAULT 1 | 是否同步加入错题本 |
| created_at | TEXT | DEFAULT now | |

### review_sessions — 回顾会话记录

Phase 3 自动回顾的会话记录。

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK AUTOINCREMENT | |
| user_id | INTEGER | NOT NULL FK | |
| unit_id | TEXT | NOT NULL | |
| weak_point_ids | TEXT | NOT NULL | 涵盖的弱点ID(JSON数组) |
| total_items | INTEGER | NOT NULL | 总项目数 |
| items_correct | INTEGER | DEFAULT 0 | 正确数 |
| completed | INTEGER | DEFAULT 0 | 是否完成 (0/1) |
| created_at | TEXT | DEFAULT now | |
| completed_at | TEXT | | 完成时间 |

---

## 索引

```sql
CREATE INDEX idx_unit_progress_user ON unit_progress(user_id);
CREATE INDEX idx_unit_progress_phase ON unit_progress(current_phase);
CREATE INDEX idx_quiz_attempts_unit ON quiz_attempts(user_id, unit_id);
CREATE INDEX idx_test_attempts_unit ON test_attempts(user_id, unit_id);
CREATE INDEX idx_error_book_user ON error_book(user_id);
CREATE INDEX idx_error_book_mastery ON error_book(mastery_level);
CREATE INDEX idx_error_book_due ON error_book(next_review_due);
```

---

## ER 关系

```
users (1) ──< (N) unit_progress
users (1) ──< (N) settings
users (1) ──< (N) quiz_attempts
users (1) ──< (N) weak_points
users (1) ──< (N) test_attempts
users (1) ──< (N) error_book
users (1) ──< (N) final_exam_attempts
users (1) ──< (N) review_sessions

test_attempts (1) ──< (1) error_book  (通过 source_test_attempt_id)
quiz_attempts (1) ──< (1) weak_points (同 unit+question upsert)
review_sessions (M:N) ── weak_points (通过 weak_point_ids JSON)
```
